# Sentiment Analysis Web App Frontend

This repository contains the React frontend for a simple NLP sentiment classifier. It provides a user interface to input text and displays the sentiment prediction returned by the backend API.

## Tech Stack

*   React
*   JavaScript
*   CSS

## Deployment

This frontend is deployed using [GitHub Pages](https://pages.github.com/). The build is generated using `npm run build` and hosted from the `gh-pages` branch.

## Local Setup (Optional)

1.  Navigate to the project directory.
2.  Install dependencies: `npm install`
3.  Start the development server: `npm start`
4.  The app will open in your browser at `http://localhost:3000`.

**Note:** If running locally, make sure the backend API is running and the `API_BASE_URL` in `src/App.js` points to the correct backend endpoint (e.g., `http://127.0.0.1:8000`).
